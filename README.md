# 🐍 Inqube Snake Game

This is a web-based snake game built as part of [Inqube](https://inqube.io)'s creative tech experiments.

- Built with: HTML, CSS, JS
- Inspired by: Classic Snake + Pixel Alien Aesthetic
- Features:

  - Apple-eating logic
  - Collision detection
  - Restart system
  - Digest animation
